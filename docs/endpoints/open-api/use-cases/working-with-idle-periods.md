# Working with idle periods

## Temporal units - How to?

There are three temporal units available for idle periods in the open api:

- The DAY unit: This unit can only be used together with a day picker.
![day-picker.png](/assets/day-picker.35ccbbc22cb42437ce7e1ce1415328d6c8dc51762cd3c04341c644f9de35b60d.71a4f21c.png)
- The WEEK unit: This unit can only be used together with a week picker. Only full weeks starting from Monday are supported.
![week-picker.png](/assets/week-picker.003c82397c892e78a63c31eeabfaecbd641ddc6c5fc01c63ab211d8465199497.71a4f21c.png)
- The MONTH unit: This unit can only be used together with a month picker. Only full months starting from the first are supported.
![month-picker.png](/assets/month-picker.7b999af1130f4ce7d306283ce482fe5e84252905ff28abaec35ee2cc1606b463.71a4f21c.png)
